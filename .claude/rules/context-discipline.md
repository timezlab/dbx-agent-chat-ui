# Context Discipline

Keep context tight:
- Read targeted sections, not full files — use Grep to locate first, then Read with offset/limit
- Suggest `/compact` before large implementation phases
- Suggest `/clear` when switching to an unrelated task, or after 3+ failed attempts on the same problem

Before starting any task that looks too large for one session — decompose it first. See `.agents/skills/context-discipline/` for the decomposition framework.
